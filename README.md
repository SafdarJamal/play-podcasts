# Play Podcasts

A podcast player.

## Development

To get a local copy of the code, clone it using git:

```
git clone https://github.com/SafdarJamal/play-podcasts.git
cd play-podcasts
```

Install dependencies:

```
npm install
```

Finally, you need to start a local web server. Run:

```
npm start
```
